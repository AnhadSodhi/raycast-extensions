# Windows Command Line Script Saver Changelog

## [Initial Release] - 2025-11-24

- Save Windows Command Prompt scripts with name, working directory, and multiple commands
- Execute saved scripts and view real-time output
- Edit existing scripts (name, directory, commands)
- Copy commands or full scripts to clipboard
- Delete scripts with confirmation dialog
- Smart organization: scripts sorted by last accessed time
- Non-fatal stderr handling for better error reporting
- Smooth navigation using Raycast's push/pop API
- Comprehensive keyboard shortcuts (Ctrl+Enter, Ctrl+E, Ctrl+C, Ctrl+Shift+C, Ctrl+D)
